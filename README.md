# Idle Tactical Turn-Based RPG

Base técnica inicial para un juego RPG táctico por turnos 4v4, pensado con estrategia web-first y un battle engine desacoplado de la UI.

## Objetivo actual

Preparar el repositorio para empezar a desarrollar un battle engine mínimo en TypeScript, sin implementar todavía gameplay completo, backend, PvP ni sistemas idle.

## Alcance de esta base

- Estructura de carpetas clara para dominio, combate, datos, UI y utilidades
- Tipos base del dominio y del estado de batalla
- Configuración mínima de TypeScript y tests
- Documentación de diseño para el MVP de combate

## Scripts

- `npm run dev`: ejecuta el entrypoint en modo desarrollo
- `npm run build`: compila TypeScript
- `npm test`: corre la suite de tests
